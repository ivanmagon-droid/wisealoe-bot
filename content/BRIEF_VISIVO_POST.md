| Elemento | Specifica |
|----------|-----------|
| Formato | 1080x1080px (feed Instagram) |
| Font titolo | Montserrat Bold o Poppins Bold |
| Font corpo | Montserrat Regular o Poppins Light |
| Colori brand | Verde FLP `#00843D`, Bianco `#FFFFFF`, Grigio scuro `#2D2D2D` |
| Colore accento | Oro chiaro `#C9A84C` (per CTA e highlight) |
| Stile | Pulito, professionale, minimalista. NO effetti troppo "MLM hype" |
| Logo | Logo Forever Living piccolo in basso a destra (opzionale) |
| Watermark | @wisealoe in basso a sinistra, semitrasparente |
